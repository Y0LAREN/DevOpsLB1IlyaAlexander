Контрольные вопросы

1. Что такое Docker и зачем он нужен?
Docker — это платформа для контейнеризации приложений, которая позволяет запускать их в изолированной среде на любом компьютере без конфликта зависимостей.

2. Что такое Dockerfile?
Dockerfile — это текстовый файл, содержащий инструкции для сборки Docker-образа, включая установку зависимостей, копирование файлов и запуск команд внутри контейнера.

3. Что такое docker-compose.yml?
Это конфигурационный файл для Docker Compose, который позволяет описывать и запускать многоконтейнерные приложения с настройкой сетей, томов и зависимостей между контейнерами.

4. Как запускать контейнеры на Mac через Docker?
Используется терминал Mac с командой `docker-compose up -d` (для запуска в фоне) после настройки Docker Desktop для Mac.

5. Как подготовить проект для GitHub?
Через команды `git init`, `git add .`, `git commit -m "initial commit"`, `git branch -M main`, `git remote add origin <URL>` и `git push -u origin main`.
